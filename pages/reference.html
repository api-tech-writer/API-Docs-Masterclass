Complete API Reference

Comprehensive documentation for all endpoints, parameters, and responses

üåê Base URL

https://api.globalweather.com/v1

All API requests must be made to this base URL using HTTPS.

üîê Authentication

All API requests require authentication using an API key in the header:

Authorization: Bearer YOUR_API_KEY

Example Request with Authentication:

curl -X GET "https://api.globalweather.com/v1/weather/current?city=London" \
  -H "Authorization: Bearer gw_live_abc123def456" \
  -H "Accept: application/json"

‚òÄÔ∏è GET /weather/current

Returns current weather conditions for a specified location.

Parameters

Name Type In Required Description
city string query No* City name (e.g., "London")
lat float query No* Latitude coordinate (-90 to 90)
lon float query No* Longitude coordinate (-180 to 180)
units string query No Temperature units: metric (default), imperial, kelvin
lang string query No Language code (ISO 639-1)

* Either city or both lat and lon are required

Request Example

curl -X GET "https://api.globalweather.com/v1/weather/current?city=Paris&units=metric" \
  -H "Authorization: Bearer YOUR_API_KEY"

Response Example

{
  "location": {
    "city": "Paris",
    "country": "FR",
    "coordinates": {
      "lat": 48.8566,
      "lon": 2.3522
    },

  },
  "current": {
    "timestamp": "2025-01-01T12:00:00Z",
    "temperature": 15.5,
    "feels_like": 14.2,
    "humidity": 72,
    "pressure": 1013,
    "visibility": 10000,
    "wind": {
      "speed": 5.2,
      "direction": 230,
      "gust": 8.1
    },
    "weather": {
      "id": 802,
      "main": "Clouds",
      "description": "scattered clouds",
      "icon": "03d"
    },

}

Error Responses

Status Code Description
400 Bad Request - Invalid parameters
401 Unauthorized - Invalid or missing API key
404 Not Found - Location not found
429 Too Many Requests - Rate limit exceeded
500 Internal Server Error

üìÖ GET /weather/forecast

Returns weather forecast for up to 7 days.

Parameters

Name Type Required Description
city string No* City name
days integer No Number of days (1-7, default: 5)
units string No Temperature units

Request Example

curl -X GET "https://api.globalweather.com/v1/weather/forecast?city=Tokyo&days=3" \
  -H "Authorization: Bearer YOUR_API_KEY"

Response Example

{
  "location": {
    "city": "Tokyo",
    "country": "JP"
  },
  "forecast": [
    {
      "date": "2025-01-02",

‚ùå Error Handling

All errors return a consistent JSON structure:

{
  "error": {
    "code": "INVALID_LOCATION",
    "message": "The specified location could not be found",
    "details": {
      "city": "InvalidCity",
      "suggestion": "Please check the spelling or try searching first"
    }
  },
  "timestamp": "2025-01-01T12:00:00Z",
  "request_id": "req_abc123"
}

Common Error Codes

Code Description Resolution
INVALID_API_KEY API key is invalid or expired Check your API key
RATE_LIMIT_EXCEEDED Too many requests Wait before retrying
INVALID_LOCATION Location not found Verify location name
SERVICE_UNAVAILABLE Temporary service issue Retry with exponential backoff
